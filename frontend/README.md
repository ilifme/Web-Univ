# Frontend - Admin Dashboard

React + Vite frontend untuk Universitas Purbayan Admin Dashboard.

## 🚀 Quick Start

\\\ash
npm install
npm run dev
\\\

Akan berjalan di \http://localhost:5173\

## 📁 Project Structure

\\\
src/
├── components/    - Reusable React components
├── pages/         - Page components
├── services/      - API calls & axios setup
├── hooks/         - Custom React hooks
├── context/       - React Context API (Auth, Global State)
├── assets/        - Images, icons, etc
├── styles/        - CSS files
├── App.jsx        - Main app component
└── main.jsx       - Entry point
\\\

## 🔐 Authentication

Token JWT disimpan di localStorage dan dikirim sebagai Bearer token di header.

## 🛠️ Available Scripts

- \
pm run dev\ - Start development server
- \
pm run build\ - Build for production
- \
pm run preview\ - Preview production build
- \
pm run lint\ - Run ESLint

## 📦 Key Dependencies

- **react** - UI framework
- **react-router-dom** - Client-side routing
- **axios** - HTTP client
- **zustand** - State management
- **react-quill** - Rich text editor
- **tailwindcss** - Styling

## 🎨 Styling

Using Tailwind CSS for styling. Edit \	ailwind.config.js\ to customize theme.
